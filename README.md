# PDF Forge — 本地 PDF 编辑器

> 🚀 **三轮对话直出的生产级 PDF 编辑器**
>
> 从需求到成品仅用 3 轮 AI 对话完成开发：核心功能 → 单文件封装 → UI 美化，全程 Vibe Coding 零手写代码。

一个完全在浏览器本地运行的 PDF 编辑工具，**单 HTML 文件**即可使用，无需服务器、无需安装、无需上传文件，所有操作均在本地完成。

采用 Anthropic 品牌风格设计，界面简洁优雅，支持常见的 PDF 页面编辑操作。

---

## ✨ 核心功能

### 页面操作
- 📂 **打开 PDF**：支持同时打开多个 PDF 文件，或直接拖放文件到界面
- 🗑️ **删除页面**：选中后一键删除，支持多选批量删除
- ➕ **插入页面**：从其他 PDF 选择页面插入到指定位置（前/后/开头/末尾）
- 🔀 **拖动排序**：可视化拖拽页面卡片调整顺序，支持多页一起拖动
- 🔄 **旋转页面**：90° 旋转选中页面，缩略图实时更新
- 📋 **复制页面**：在原位置后复制选中页面
- 📤 **提取页面**：将选中页面导出为独立的新 PDF
- ↔️ **反转顺序**：反转选中页面的排列顺序
- 💾 **导出 PDF**：合并所有页面生成最终 PDF 文件

### 交互特性
- ⌨️ 完整快捷键支持
- 🖱️ 缩略图缩放控制（50%~200%）
- 📁 侧边栏可调整宽度
- 📊 实时统计（总页数/选中页数/文件数）
- 🔔 操作结果 Toast 提示
- 🎨 优雅的 Anthropic 品牌风格 UI，纸张纹理质感

## 🚀 使用方法

### 直接使用
1. 下载 `pdf-forge.html` 文件到本地
2. 双击用浏览器打开即可使用（推荐 Chrome/Edge/Firefox 等现代浏览器）
3. 首次打开需要联网加载 PDF.js 和 PDF-lib 依赖（通过 CDN），之后浏览器会缓存，离线也可使用

### 分享给他人
直接把 `pdf-forge.html` 单文件发给对方即可，对方双击打开就能用，不需要任何额外安装。

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| 点击页面 | 选中单个页面 |
| `Shift + 点击` | 多选连续页面 |
| `Delete` / `Backspace` | 删除选中页面 |
| `Ctrl + A` | 全选/取消全选 |
| `Ctrl + S` | 导出 PDF |
| `Esc` | 取消选中 / 关闭弹窗 |
| 拖动页面 | 调整页面顺序 |

## 🛠️ 技术栈
- **PDF 渲染**：[PDF.js](https://mozilla.github.io/pdf.js/)
- **PDF 处理**：[PDF-lib](https://pdf-lib.js.org/)
- **纯前端**：HTML + CSS + 原生 JavaScript，无框架依赖
- **单文件**：所有代码内联在一个 HTML 文件中，总大小不到 60KB
